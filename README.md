# QA Automation Engineer Learning Roadmap

## Phase 1 — Testing Fundamentals

Learn the core QA concepts first.

### Topics
- What is Software Testing
- Manual Testing Basics
- SDLC & STLC
- Bug Life Cycle
- Test Case Writing
- Severity vs Priority
- Functional vs Non-functional testing
- Smoke Testing
- Regression Testing
- Sanity Testing
- UI Testing
- API Testing Basics
- Cross-browser Testing

### Practice
- Test login page
- Test signup form
- Test cart system
- Write test cases in Google Docs or Notion
